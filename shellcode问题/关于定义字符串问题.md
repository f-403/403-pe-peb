### 全局定义
```css
//如果你是如下定义：
char name[] = {'a', 'b', 'c'}
//或如下定义:
char name[] = "abc"

//定义为全局变量的话， 都会存在.data区，如果你要放指定区段里， 要加如下参数才行:
#pragma code_seg("ABCDE");
__declspec(allocate("ABCDE")) char name[] = { 'a', 'b', 'a'};
__declspec(allocate("ABCDE")) char name[] = "abc";

//或
#pragma code_seg(".text");
__declspec(allocate(".text")) char name[] = { 'a', 'b', 'a'};
__declspec(allocate(".text")) char name[] = "abc";

```

## 内部变量
```css
//如果你是如下定义：
char name[] = {'a', 'b', 'c', '\0'}
它会存在 .text段或你指定的段


//如果你是合起来定义:
```
